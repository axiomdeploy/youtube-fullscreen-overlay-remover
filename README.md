# YouTube Fullscreen Overlay Remover

A lightweight JavaScript bookmarklet that removes overlay UI elements in YouTube fullscreen mode for a clean, distraction-free viewing experience.

## Features

- Hides fullscreen overlay UI elements
- Automatically detects fullscreen mode
- Minimal and fast execution
- No installation required

## Usage

1. Open the `yt-fullscreen-overlay-remover.js` file.
2. Copy the script.
3. Create a new bookmark in your browser.
4. Paste the script into the bookmark URL field.
5. Open a YouTube video and enter fullscreen.
5. Click the bookmark to activate it.

## How It Works

This script injects a small CSS rule that hides YouTube overlay elements only when fullscreen mode is active. It listens for fullscreen changes and toggles a class on the page to control visibility.

## Notes

- Best used in modern browsers such as Chrome or Edge
- Must be triggered manually through a bookmarklet
- Does not permanently modify YouTube
